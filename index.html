Questions?

Can I pay with PayPal?

Sure! Just transfer the appropriate amount for the licence you'd like to purchase to gavin@gavinballard.com, and get in touch with me to arrange delivery.

Which licence is best for me?

If you're just looking to get your feet wet with Shopify and Bootstrap, or want to take Bootstrap for Shopify for a test drive, I'd recommend starting out with an Affiliate Licence — you can always upgrade to a Single or Unlimited Licence later.

If you'd prefer not to use an Affiliate Licence (for example, if you've already signed up for Shopify), jump in to a Single Licence. If you later decide you'd like to upgrade to an Unlimited Licence, you'll only pay the difference for the upgrade.

If you'd like to sell the themes you build with Bootstrap for Shopify (on the Shopify Theme Store, to multiple clients, or otherwise), you'll need an Unlimited Licence.

How long will support / updates last for?

I provide email support for anyone using the framework — unless you're emailing me ten times a day, I'm happy to answer any and all questions! Bootstrap for Shopify will be updated as Bootstrap is updated — the current plan is to provide free updates for all future 2.x and 3.x versions.

What if I'm not satisfied?

It's the old mantra - I'm not happy unless you're happy! If you've purchased Bootstrap for Shopify but find it lacking, I'll do everything I can to resolve your issues. However, if that doesn't work out, I'll refund you the full purchase price.

How customisable is it?

Shopify templates and other frameworks often come with custom HTML, CSS and Javascript you don't want, need or understand. Bootstrap for Shopify is 100% vanilla Bootstrap, with no added styles or markup other than those essential to operating on the Shopify platform.

Don't want to use jQuery? Not a problem. Got your own custom Bootstrap build? Drop it in.

Why are you charging for this?

Assuming your time is valuable (and I think it is!), anything that saves you a few hours or more on your next project can be a huge benefit. Bootstrap for Shopify takes a lot of the drudge work out of getting a clean slate ready for you to work your magic.

Moreover, you also get a framework designed for building clean & maintainable Shopify Themes, with a large number of helpful snippets, patterns and techniques. That's not even mentioning the really helpful Getting Started Guide, which will put you right at home building with Bootstrap and Shopify.
